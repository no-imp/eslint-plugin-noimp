# eslint-plugin-imp

The original eslint `imp` slayer. Beware those imps ... use `eslint-plugin-imp` to in kill mode `2` to avoid certain death.

## Usage

```
{
  "extends": "eslint:recommended",
  "plugins": [
    "eslint-plugin-noimp"
  ],
  "rules": {
    "noimp/killall": 2
  }
}
```
